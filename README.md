# Mental Health Assistant

This project was created as part of the Google AI Agents 5-Day Program Capstone Project.  
The Mental Health Assistant is designed to support users by recognizing mental health conditions, emotional states, and providing comforting, positive responses.  
It acts as a companion that understands user input and helps promote better mental well-being using AI-driven interactions.

## Key Features
- Identifies emotional tone from user responses
- Provides supportive and motivational feedback
- Aims to reduce negative mental states through conversation
- Can help guide users toward healthier thinking patterns

## Technologies Used
- Python
- Google AI Agents API

## Future Improvements
- More advanced emotional analysis
- Connecting with mental health resources
- Multi-language support and real-time interaction
